## The Dashboard

The dashboard is your home base in Elmo. It gives you a high-level view of your brand's AI visibility.

## Key Metrics

The dashboard displays several key metrics at the top:

| Metric | Description |
|--------|-------------|
| **Visibility Score** | Percentage indicating how often your brand appears in AI responses |
| **Prompts Tracked** | Total number of active prompts being monitored |
| **Evaluations** | Total number of AI model queries processed |
| **Run Frequency** | How often the worker queries AI models |
| **Last Updated** | When the most recent evaluation completed |

## Visibility Trends

A chart shows your visibility score over the past 30 days. This helps you spot trends — is your AI visibility improving, declining, or stable?

## Waiting for Data

If the worker hasn't run yet, the dashboard shows a "Waiting for First Evaluation" state. The worker queries AI models on the schedule configured during setup. Your first results should appear within a few minutes of starting Elmo.

## Navigation

Use the sidebar to navigate between sections:

- **Overview** — The dashboard described on this page
- **Visibility** — Detailed per-prompt visibility data
- **Citations** — Citation breakdown by category
- **Settings** — Brand, competitors, prompts, and LLM configuration
